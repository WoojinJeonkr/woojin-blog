## 1. Problem

## 2. Answer

```py
# 정수 A, B, C를 각각 입력
A = int(input().strip())
B = int(input().strip())
C = int(input().strip())

# 세 수의 합이 21 이하이면 1, 그렇지 않으면 0을 출력
print(1 if A + B + C <= 21 else 0)
```
