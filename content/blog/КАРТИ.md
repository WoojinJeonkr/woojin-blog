## 1. Problem

해당 문제는 [여기](https://www.acmicpc.net/problem/24347)에서 확인하실 수 있습니다.

## 2. Answer

```py
import sys

nums = list(map(str, sys.stdin.readline().split()))

# 정렬 기준: a+b vs b+a 비교
nums.sort(key=lambda x: x * 4, reverse=True)

# 결과 출력
result = ''.join(nums)

# 예외 처리: 모두 0인 경우
if result[0] == '0':
    print(0)
else:
    print(result)
```
