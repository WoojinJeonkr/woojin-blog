## 1. Problem

해당 문제는 [여기](https://www.acmicpc.net/problem/17614)에서 확인하실 수 있습니다.

## 2. Answer

```py
# 사용자로부터 정수 N 입력
N = int(input())

# 박수의 총 횟수를 저장할 변수 초기화
count = 0

# 1부터 N까지 모든 수를 차례대로 확인
for i in range(1, N + 1):
    # 현재 숫자를 문자열로 변환하여 각 자릿수를 하나씩 확인
    for ch in str(i):
        # 자릿수가 3, 6, 9 중 하나라면 박수 횟수 증가
        if ch in ('3', '6', '9'):
            count += 1

# 지금까지 센 박수의 총 횟수 출력
print(count)
```
