## 2. 프로젝트 폴더 구조 설계

Antigravity 에이전트가 이해하기 쉽고 추후 유지보수가 용이하도록 표준적인 폴더 구조를 잡았습니다.

```txt
Antigravity-LifeGraph/
├── uploads/            # 사용자가 업로드한 이력서 임시 저장
├── app.py              # 전체 서비스를 구동할 메인 파일
├── parser.py           # Word, PPT 파일에서 텍스트를 추출하는 로직
├── analyzer.py         # Hugging Face 모델 연동 및 데이터 분석
├── utils.py            # 이미지 저장 및 클립보드 복사 등 유틸리티
└── requirements.txt    # 프로젝트 필요 라이브러리 목록
```

## 3. 의존성 관리: requirements.txt 작성

파이썬 프로젝트를 진행할 때는 내가 사용한 외부 라이브러리 목록을 기록해두는 것이 중요합니다. 이를 위해 requirements.txt 파일을 작성합니다.

- **requirements.txt**가 필요한 이유
  - 환경 복구: 다른 컴퓨터나 서버에서 내 프로젝트를 실행할 때, pip install -r requirements.txt 명령 한 번으로 필요한 라이브러리를 똑같이 설치할 수 있습니다.
  - 버전 관리: 팀 프로젝트나 배포 시 라이브러리 버전 충돌을 방지합니다.

이제 4번 단계에서 작성할 코드에서 사용할 라이브러리를 **requirements.txt**에 아래와 같이 작성해둡니다.

```txt
python-docx
python-pptx
```

## 4. 비정형 데이터에서 텍스트 추출

이력서는 사람마다 형식이 제각각입니다. 어떤 사람은 표를 쓰고, 어떤 사람은 PPT의 텍스트 박스를 활용하죠. 이 모든 데이터에서 텍스트를 긁어오기 위해 python-docx와 python-pptx 라이브러리를 사용하고자 합니다.

이제 Antigravity에게 Fast 모드로 설정하고 아래와 같이 질문을 던져 **parser.py** 파일을 작성해보겠습니다.

### Agent Prompt

```txt
사용자가 업로드한 .docx와 .pptx 파일에서 텍스트를 추출하고 싶어. @files:requirements.txt에 기재한 라이브러리를 사용해서, 특정 파이썬 버전의 환경이 아닌 범용적인 파이썬 환경에서 빈 줄은 제외하고 텍스트만 리스트로 반환하는 함수를 `@files:parser.py`에 작성해줘. 단, py 파일 코드의 주석은 한글로 작성해줘
```

### Antigravity 팁: @files 참조 기능

`@files:파일명`이라고 입력하면 에이전트에게 해당 파일의 위치와 내용을 명시적으로 알려줄 수 있습니다. 만약 같은 이름의 파일이 여러 폴더에 있더라도 혼동 없이 정확한 파일을 참조(cascade)하게 만드는 강력한 기능입니다. 커서를 파일명 끝에 두고 Enter를 누르면 활성화됩니다.

에이전트는 제 요청을 듣고 즉시 아래와 같은 코드를 생성해 주었습니다.

```py
import os
import docx
from pptx import Presentation

def extract_text(file_path):
    """
    다양한 파일 형식(.docx, .pptx)에서 텍스트를 추출합니다.
    각 문자열이 단락 또는 의미 있는 텍스트 블록인 문자열 리스트를 반환합니다.
    빈 줄은 제외됩니다.
    """
    if not os.path.exists(file_path):
        return []

    ext = os.path.splitext(file_path)[1].lower()

    if ext == '.docx':
        return extract_text_docx(file_path)
    elif ext == '.pptx':
        return extract_text_pptx(file_path)
    else:
        # 지원되지 않는 형식이거나 폴백(fallback) 처리
        return []

def extract_text_docx(file_path):
    """
    .docx 파일에서 텍스트를 추출합니다.
    """
    try:
        doc = docx.Document(file_path)
        text_list = []
        for paragraph in doc.paragraphs:
            text = paragraph.text.strip()
            if text:
                text_list.append(text)
        return text_list
    except Exception as e:
        print(f".docx 파싱 중 오류 발생: {e}")
        return []

def extract_text_pptx(file_path):
    """
    .pptx 파일에서 슬라이드별로 텍스트를 추출합니다.
    """
    try:
        prs = Presentation(file_path)
        text_list = []
        for slide in prs.slides:
            for shape in slide.shapes:
                if hasattr(shape, "text") and shape.text.strip():
                    # 텍스트를 포함한 도형에서 텍스트 추출
                    text = shape.text.strip()
                    if text:
                        text_list.append(text)
                elif hasattr(shape, "text_frame"):
                    # 텍스트 프레임이 있는 도형의 경우 모든 단락 수집
                    for paragraph in shape.text_frame.paragraphs:
                        text = paragraph.text.strip()
                        if text:
                            text_list.append(text)
        return text_list
    except Exception as e:
        print(f".pptx 파싱 중 오류 발생: {e}")
        return []
```
