## 1. Problem

해당 문제는 [여기](https://www.acmicpc.net/problem/14730)에서 확인하실 수 있습니다.

## 2. Answer

```py
# 항의 개수 입력
N = int(input())

result = 0

# 각 항에 대해 처리
for _ in range(N):
    C, K = map(int, input().split())
    # 미분 후 x=1에서의 값은 C * K
    result += C * K

# f'(1) 출력
print(result)
```
