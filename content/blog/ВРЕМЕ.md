## 1. Problem

해당 문제는 [여기](https://www.acmicpc.net/problem/31243)에서 확인하실 수 있습니다.

## 2. Answer

```py
import sys  # 표준 입력을 빠르게 받기 위해 sys 모듈 사용

times = []  # 각 자동차의 이동 시간을 저장할 리스트

# 3개의 자동차 정보 입력 받기
for _ in range(3):
    sh, sm, eh, em = map(int, sys.stdin.readline().split())  # 출발 시각(sh, sm), 도착 시각(eh, em)

    start = sh * 60 + sm  # 출발 시간을 분 단위로 변환
    end = eh * 60 + em    # 도착 시간을 분 단위로 변환

    duration = end - start  # 이동 시간 계산

    # 자정을 넘어간 경우 (음수일 때) 하루(1440분)를 더해 보정
    if duration < 0:
        duration += 1440

    times.append(duration)  # 계산된 이동 시간을 리스트에 추가

min_time = min(times)  # 최소 이동 시간
max_time = max(times)  # 최대 이동 시간

# 분 단위 시간을 "시:분" 형식으로 변환하는 함수
def format_time(t):
    h = t // 60      # 시간 계산
    m = t % 60       # 분 계산
    return f"{h}:{m:02d}"  # 분은 두 자리로 출력 (예: 2:05)

# 최소 이동 시간 출력
print(format_time(min_time))

# 최대 이동 시간 출력
print(format_time(max_time))
```
